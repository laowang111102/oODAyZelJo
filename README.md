# 大学生请假管理系统 python697

## 项目概述

大学生请假管理系统是一款基于B/S结构的网络应用，旨在帮助高校简化请假流程，提高工作效率。系统采用Python和Django框架进行开发，前端使用了Vue.js技术，数据库选用MySQL。

## 技术栈

- 后端：Python、Django
- 前端：Vue.js
- 数据库：MySQL

## 功能模块

### 学生

- 个人中心
- 学生请假

### 辅导员

- 个人中心
- 学生请假
- 班级信息
- 请假数据
- 请假次数预测

### 管理员

- 个人中心
- 学生管理
- 辅导员管理
- 班级管理
- 请假审核
- 请假数据统计
- 请假次数预测
- 系统设置

## 系统角色

- 学生
- 辅导员
- 管理员

## 运行环境

- 服务器：兼容Python和Django的服务器环境
- 客户端：现代浏览器

## 部署步骤

1. 配置Python和Django环境
2. 安装依赖库
3. 导入MySQL数据库结构
4. 配置应用设置
5. 部署至服务器
